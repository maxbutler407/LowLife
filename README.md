# LowLife
LowLife is a C++ passion project of mine, that I made on my own during the Summer of 2024. It is a text-based RPG based in a world where sci-fi and fantasy are fused together in a gritty but fantastical world [WARNING: some images and text might be too explicit].

I am very happy with how it turned out and I hope that that others have fun playing it. Enjoy!
