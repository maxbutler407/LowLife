# LowLife
LowLife is a C++ passion project of mine, made during the Summer of 2024. It is a text-based RPG based in a world where sci-fi and fantasy are fused together in a gritty but fantastical world [WARNING: some images and text might be too explicit].

Some of the original art and audio has been changed/removed to avoid copyright issues, I would prefer to use non-AI images but that means they would likely be copyrighted.
